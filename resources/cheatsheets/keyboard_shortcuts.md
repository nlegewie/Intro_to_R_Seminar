# Tastaturkürzel

Tastaturkürzel sind Abkürzungen, mit denen du dir Navigieren und Klicks mit der Maus ersparst. Es gibt einige Kürzel, die in vielen Programmen funktionieren ([Allgemeine Tastaturkürzel](#allgemeine-tastaturkürzel)) und solche, die speziell in RStudio gelten ([RStudio-Tastaturkürzel](#rstudio-tastaturkürzel))

## Warum sich das lohnt

Tastaturkürzel wirken am Anfang wie unnötiger Aufwand — du kennst ja die Maus. Aber schon nach ein paar Tagen machen sie einen spürbaren Unterschied: weniger Klicken, weniger Suchen, weniger Warten. Über ein Semester hinweg sparst du Stunden.

Wichtiger noch: Diese Kürzel gelten nicht nur für RStudio. Die allgemeinen Kürzel (Kopieren, Einfügen, Rückgängig, Suchen) funktionieren in fast jedem Programm — Word, Excel, Browser, E-Mail, Terminals. Sie sind eine Grundfertigkeit, die dir im gesamten Studium und in vielen Jobs in der Wissenschaft, in der öffentlichen Verwaltung und in der Privatwirtschaft nützt.

**Tipp:** Lerne nicht alle auf einmal. Nimm dir jede Woche 2–3 neue Kürzel vor und benutze sie bewusst, bis sie du sie von ganz allein verwendest.

## Allgemeine Tastaturkürzel

Diese funktionieren in fast allen Programmen auf deinem Computer.

### Grundlagen

| Aktion | Windows | macOS |
|--------|---------|-------|
| Kopieren | Ctrl + C | Cmd + C |
| Ausschneiden | Ctrl + X | Cmd + X |
| Einfügen | Ctrl + V | Cmd + V |
| Rückgängig | Ctrl + Z | Cmd + Z |
| Wiederherstellen | Ctrl + Y | Cmd + Shift + Z |
| Speichern | Ctrl + S | Cmd + S |
| Alles markieren | Ctrl + A | Cmd + A |
| Suchen | Ctrl + F | Cmd + F |
| Suchen und Ersetzen | Ctrl + H | Cmd + Shift + H |


### Text markieren

| Aktion | Windows | macOS |
|--------|---------|-------|
| Wort markieren | Doppelklick auf das Wort | Doppelklick auf das Wort |
| Ganze Zeile markieren | Home, dann Shift + End | Cmd + Shift + ← dann Shift + Cmd + → |
| Von Cursor bis Zeilenanfang | Shift + Home | Cmd + Shift + ← |
| Von Cursor bis Zeilenende | Shift + End | Cmd + Shift + → |
| Wortweise markieren | Ctrl + Shift + ← / → | Option + Shift + ← / → |
| Mehrere Zeilen markieren | Shift + ↑ / ↓ | Shift + ↑ / ↓ |


### Navigation

| Aktion | Windows | macOS |
|--------|---------|-------|
| Wortweise springen | Ctrl + ← / → | Option + ← / → |
| Zeilenanfang / -ende | Home / End | Cmd + ← / → |
| Dokumentanfang / -ende | Ctrl + Home / End | Cmd + ↑ / ↓ |
| Zwischen offenen Tabs wechseln | Ctrl + Tab | Ctrl + Tab |

## RStudio-Tastaturkürzel

### Die wichtigsten (ab Tag 1 verwenden)

| Aktion | Windows | macOS |
|--------|---------|-------|
| **Aktuelle Zeile/Auswahl ausführen** | **Ctrl + Enter** | **Cmd + Enter** |
| **Gesamtes Skript ausführen (Source)** | **Ctrl + Shift + Enter** | **Cmd + Shift + Enter** |
| **R-Session neu starten** | **Ctrl + Shift + F10** | **Cmd + Shift + F10** |
| Neues Skript erstellen | Ctrl + Shift + N | Cmd + Shift + N |


### Code schreiben

| Aktion | Windows | macOS |
|--------|---------|-------|
| Zeile(n) auskommentieren / einkommentieren | Ctrl + Shift + C | Cmd + Shift + C |
| Zeile(n) nach oben / unten verschieben | Alt + ↑ / ↓ | Option + ↑ / ↓ |
| Zeile duplizieren | Ctrl + Shift + D | Cmd + Shift + D |
| Einrücken | Tab | Tab |
| Ausrücken | Shift + Tab | Shift + Tab |
| Pipe-Operator einfügen (`|>`) | Ctrl + Shift + M | Cmd + Shift + M |
| Zuweisungspfeil einfügen (`<-`) | Alt + - | Option + - |
| Autovervollständigung | Tab | Tab |


### Navigation in RStudio

| Aktion | Windows | macOS |
|--------|---------|-------|
| Zwischen offenen Dateien wechseln | Ctrl + Tab | Ctrl + Tab |
| Gehe zu Zeile | Ctrl + G | Cmd + G |


### R Markdown

| Aktion | Windows | macOS |
|--------|---------|-------|
| Aktuellen Chunk ausführen | Ctrl + Shift + Enter | Cmd + Shift + Enter |
| Neuen Code-Chunk einfügen | Ctrl + Alt + I | Cmd + Option + I |
| Dokument rendern (Knit) | Ctrl + Shift + K | Cmd + Shift + K |
