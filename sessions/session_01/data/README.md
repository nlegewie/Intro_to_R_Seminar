# Daten für Session 1 (Endgame-Demo)

Die Datei **`owid_data.csv`** ist eine **reduzierte Auszugsdatei** aus dem vollständigen Our-World-in-Data-Aufbereitungssatz: nur Länder mit bekannter `world_region`, Jahre 1990–2020 und die Spalten, die die Demo braucht. So bleibt das Repository klein und die Demo läuft ohne manuellen Datenimport.

Wenn du den **vollen** Datensatz verwenden willst, ersetze diese Datei durch deine `owid_data.csv` aus der Datenaufbereitung — `demo_endgame.Rmd` erwartet denselben Dateinamen und dieselben Spaltennamen.
