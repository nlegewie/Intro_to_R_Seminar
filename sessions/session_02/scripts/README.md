# Skripte zu Session 2

In Session 2 arbeiten wir **noch ohne externe `.R`-Skripte** — alle Daten werden direkt im R Markdown als Vektoren eingetippt, kein `read_csv()` und kein `source()` notwendig.

Das Auslagern von Code in separate Skripte ist eine saubere Praxis, die du ab **Session 3** (Datentransformation) lernst und im Abschlussbericht anwendest.
