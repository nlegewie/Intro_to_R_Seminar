# Daten für Session 3

Die Datei `toy_data_session_03.csv` ist ein kleiner Übungsdatensatz für die Sitzung.

## Inhalt

- 20 Länder
- Jahr: 2015
- Variablen:
  - `country`
  - `year`
  - `region`
  - `gdp_per_capita`
  - `life_expectancy_birth`
  - `child_mortality_rate`

## Zweck

Die Datei ist bewusst klein gehalten, damit du dich auf den Workflow konzentrieren kannst:

1. CSV einlesen
2. Datensatz erkunden (`glimpse()`, `nrow()`, `names()` usw.)
3. erste Histogramme erstellen

Für Hausaufgaben und den Abschlussbericht arbeitest du später im Kernprojekt mit dem größeren Datensatz.
